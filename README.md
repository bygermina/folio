## Installation and Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The project is automatically deployed to GitHub Pages:

```bash
npm run deploy
```
